MODULE_PAUSED, CAP_EXCEEDED, RATE_LIMIT_EXCEEDED, ORACLE_UNHEALTHY,
NO_VALID_ROUTE, SLIPPAGE_EXCEEDED, ACCESS_DENIED, NOT_FOUND, UNSUPPORTED_CHAIN

Shape:
{ "domain":"NETWORK", "code":"TIMEOUT", "message":"...", "cause":"..." }
